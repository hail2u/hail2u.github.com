About

現在は以下のような点を念頭に入れて作成しています。

  • 欲しいと思ったら作る

以下のFirefox Extensionへの感想や要望、バグ・リポート等はフィードバック・フォームでお報せください。

Bookmark All

Bookmark all opening tabs quickly without any dialog.

現在開いている全てのタブを、一発でまとめてブックマークします。フォルダを指定するパス(階層)やフォルダ名は自由に指定できるので

Firefoxの機能の一つである「すべてのタブをブックマーク」で、毎回フォルダ名を指定するのがわずらわしいので作りました。

Install

Bookmark Allをインストール (Add-ons for Firefoxに移動します)

Usage

「ツール」メニューにある「Bookmark All」というメニュー項目から実行するか、Ctrl+Alt+Dを同時に押してください。

ツールバーボタンも用意してあるので、ツールバーのカスタマイズから追加すればボタン一発で実行できます。

Settings

フォルダを作成するパス
フォルダを作成するパスをスラッシュを区切り文字として指定します。値を指定しない場合は最上層に作成されます。
フォルダ名の頭に付ける文字列
フォルダ名の頭に付ける固定の文字列を指定します。後ろに日付けと時刻が必ず付きます。

Context Style Switcher

Change Page Style via context menu or toolbar button.

コンテント・エリア上でのコンテキスト・メニューに、スタイルシートの切り替えメニューを追加するFirefoxアドオンです。

いつのまにかメイン・メニューからしかスタイル切り替えができなくなっていたので、手軽に切り替えられるように作りました。

Install

Context Style Switcherをインストール (Add-ons for Firefoxに移動します)

Usage

コンテント・エリア上で右クリックしてください。すると「スタイルシート(Y)」というメニュー項目が追加されているので、そこから任意のStyleを選択するだけです。

Settings

設定項目はありません。

Open Profile Folder

Open current profile folder from "Tools" menu or toolbar button.

現在のプロファイルのフォルダを開くFirefoxアドオンです。

ヒマだったので作りました。

Install

Open Profile Folderをインストール (Add-ons for Firefoxに移動します)

Usage

「ツール」メニューにある「プロファイル・フォルダを開く...」というメニュー項目から実行してください。

Settings

以下に指定したアプリケーションをフォルダを開くの使う
好みのアプリケーションでフォルダを開く機能を有効にするか無効にするかを指定します。オフにするとシステムのデフォルトのファイラーで開かれます(Linuxではうまく動きません)。
アプリケーションのパス
フォルダを開くアプリケーションのパスをフルパスで指定します。
起動時の引数
フォルダを開くアプリケーションにプロファイル・フォルダのパスと同時に渡す引数を指定します。プロファイル・フォルダのパスは引数の最後に加えられます。

Places in Tab

Display your bookmarks or pages you've viewed recently in tab instead of window like Safari.

通常はウィンドウで開いてしまう「履歴とブックマークの管理」をSafariのようにタブで開くFirefox Extensionです。

Install

Places in Tabをインストール (Add-ons for Firefoxに移動します)

Usage

「ブックマーク」メニューにある「ブックマークの管理をタブに表示」というメニュー項目または「履歴」メニューにある「すべての履歴をタブに表示」というメニュー項目から実行するか、Ctrl+Alt+BまたはCtrl+Alt+Hを同時に押してください。

メニュー項目やツールバーボタンをホイールクリックすると新しいタブで開きます。

Settings

デフォルトのメニュー項目を隠す
チェックを入れると「履歴」メニューや「ブックマーク」メニューに元々ある、ウィンドウで「履歴とブックマークの管理」を開くメニュー項目を非表示にします。

Reload Tab On Double-Click

Reload tab when double-clicking on tab.

タブ上でダブルクリックすると、そのタブで表示しているページをリロードするFirefox Extensionです。

ヒマだったので作りました。

Install

Reload Tab On Double-Clickをインストール (Add-ons for Firefoxに移動します)

Usage

タブ上で左ダブル・クリックしてください。それだけです。Shiftを押しながら左ダブル・クリックした場合はキャッシュを無視してリロードします。

Settings

設定項目はありません。
