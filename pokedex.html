pokedex.json の利用サンプル

ポケモンの基本的な能力をJSON化したデータを利用したサンプル。

各ポケモンのデータ

{
  "name"     : "フシギダネ",
  "type"     : ["くさ", "どく"],
  "hp"       : 45,
  "attack"   : 49,
  "defence"  : 49,
  "sattack"  : 65,
  "sdefence" : 65,
  "speed"    : 45,
  "ability"  : ["しんりょく"]
}

typeとabilityは必ずArray。

ポケットモンスター・ポケモン・Pokémonは任天堂・クリーチャーズ・ゲームフリークの登録商標です
