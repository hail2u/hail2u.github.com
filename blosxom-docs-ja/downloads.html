ダウンロード

試しにBlosxomを手に入れようと決めたのですね。私と同様に好きになると思いますよ。

Blosxom...

貴方に最も合うBlosxomを選んで下さい。

Mac OS X

Mac OS X用Blosxomインストーラーバージョン2.0[.pkg.tar.gz]は手作業で何らインストールしなくてもBlosxomを手に入れられるものです。マウスの数回のクリックで面倒なこと無く、Blosxom本体、幾つかのサンプルフレーバー、ドキュメントそして便利なプラグインがインストールされます。貴方がパッケージアプリケーションを使うことに慣れていて、手っ取り早く行いたいのであればきっと便利なはずですよ！

もしかすると貴方は自分でやりたい人で、Blosxomがどう動くかいじりたいかも知れません。なら、スクリプトをダウンロードして適切なディレクトリを作り、設定をいじり倒しても良いでしょう。

Windows 2000/XP

Windows 2000/XP用Blosxomインストーラーは現在準備中です。常にBlosxomニュースをチェックして下さい。もちろんメーリングリストも。まずは"全ての人へ"バージョンを使って下さい。

全ての人へ

Blosxomバージョン2.0[ZIP]はある場所へドロップして少し環境設定をするだけの簡単なCGIスクリプトです。入手したらインストールの説明を読んで下さい。

フレーバー...

Blosxomのフレーバーテンプレートは貴方のBlosxomウェブログの見た目を完全にカスタマイズするものです。フレーバーの詳細をオンラインで読むことができます。

  • Blosxom Flavour Sampler [ZIP]はサンプルのBlosxomフレーバーのコレクションで突付いたりとか色々できます。これを解凍して貴方のBlosxomのデータディレクトリにドラッグして下さい。

プラグイン...

Blosxomを拡張したり統合したりできるたくさんのサードパーティー製プラグインがあります。プラグイン登録ページで全てを見つけることができます。
